Aperçu du projet

Cette application de gestion de cours a été développée avec le framework Symfony et utilise une base de données MySQL. Son objectif est de permettre à l'administrateur de gérer les étudiants, les salles de cours, et les professeurs de manière efficace.

Les professeurs sont associés à une salle de cours, tandis que les étudiants sont liés à une discipline. L'administrateur peut gérer ces associations et effectuer des actions telles que l'ajout d'un nouvel étudiant, la modification des détails d'un professeur, ou la consultation des informations d'une salle de cours.

Cette application offre une interface conviviale pour gérer les composants clés d'une institution éducative, en utilisant les fonctionnalités puissantes du framework Symfony et une base de données MySQL pour stocker les informations essentielles.
